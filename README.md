# Алгоритм шифрування Цезаря мовою C++

## Опис проєкту
Даний проєкт є курсовою роботою, присвяченою реалізації **алгоритму шифрування Цезаря** мовою програмування **C++**.  
Програма дозволяє зашифровувати та розшифровувати текст шляхом циклічного зсуву символів латинського алфавіту на задану кількість позицій.

Алгоритм працює окремо з:
- великими літерами (`A–Z`);
- малими літерами (`a–z`);

Інші символи (пробіли, цифри, знаки пунктуації) не змінюються.

---

## Алгоритм Цезаря
Шифр Цезаря — це один із найпростіших методів симетричного шифрування.  
Суть алгоритму полягає в заміні кожної літери повідомлення на іншу, що знаходиться на фіксовану кількість позицій далі в алфавіті.

Наприклад, при зсуві **3**:
- `A → D`
- `b → e`
- `Z → C`

---

## Функціональність програми
- Введення довільного тексту з клавіатури
- Введення величини зсуву
- Шифрування тексту
- Дешифрування зашифрованого тексту
- Підтримка від’ємного та великого зсуву (через операцію `% 26`)

---

## Структура коду
- `caesarEncrypt()` — функція шифрування тексту
- `caesarDecrypt()` — функція дешифрування (через зворотний зсув)
- `main()` — введення даних та виведення результатів
